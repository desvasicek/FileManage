# File Opening
## Can Open...
- .zip
- .png
- .jpg
- .jpeg
- .mp3
- .mp4
- .vid
- .wav
- .ogg
- .exe
## Will Open On...
- Image Files: Image Viewer
- ZIP File: Xarchiver
- EXE files: command line (shell)
- Audio and Video Files: VLC
